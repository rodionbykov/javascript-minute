Event Listeners

Any Element can send (dispatch, emit) Event, that can be listened to by other Element. Upon receiving Event, Element can do some action.

To react upon receiving Event, an Element should have registered Event Listener function. This is also called 'binding', where function is bound to an Element.

In this example, when button element is clicked, it dispatches 'click' event.

In the same time, button listens to 'click' event, and has registered event handler, so when event is received, button reacts by calling clickHandlerFuction.

        const btn1 = document.querySelector("#btn1");

        btn1.addEventListener('click', clickHandlerFuction);

        function clickHandlerFuction() {
            alert('It was clicked!');
        }
