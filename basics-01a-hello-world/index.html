To run some Javascript in browser, just open browser console (press F12) and type some Javascript statements. Console will immediately show the result.

    1+2
    > 3

We will load Javascript files using <script> tag.

      <script src="./helloworld.js"></script>
    </body>

Usually Javascript are loaded at the end of page, so it is good practice your <script> tag right before closing <body> tag.

Use console.log method to output a variable or result of the statement into console.

    console.log("Hello, World!");
